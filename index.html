Project: HPE Aruba Networking

HTML5 CSS JavaScript PHP WordPress CMS SEO Google Tag Manager Google Analytics (GA4) Looker Studio

In my time at Aruba, the website arubanetworks.com grew into a global, lead-generating marketing engine supporting 17 languages. I have seen it through rebranding (twice), worked on 3rd-party integrations, optimized page performance, managed analytics tracking, created traffic reports and dashboards. I'm proud of my contribution to the website success, but it has been sunsetted in November 2024 and much of it is no longer online. Please see below for samples of my work.

Website

I have developed many of the site's templates, themes, and components (written in PHP, interactions in JavaScript). Showing the 2024 rebranding.

Landing pages

Product wizard

Navigation menu
